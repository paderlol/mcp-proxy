# Code of Conduct

We want this project to be a welcoming, productive space. In short:

- **Be respectful.** Disagree with ideas, not people.
- **Be constructive.** If you have a criticism, pair it with a suggestion.
- **Assume good intent.** Most conflicts come from miscommunication, not malice.
- **Respect scope.** This is a secret-management utility for MCP — keep discussions relevant.

## Unacceptable behavior

- Harassment, personal attacks, or discriminatory language based on identity (race, gender, sexual orientation, disability, religion, nationality, etc.)
- Deliberate intimidation, stalking, or doxxing
- Sustained disruption of discussions
- Publishing others' private information without permission

## Reporting

If you see or experience behavior that violates this code, please open a private GitHub Security Advisory, or email the maintainer listed in the repository's profile. Reports will be reviewed in good faith.

## Enforcement

Maintainers may warn, temporarily block, or permanently ban contributors whose behavior violates this code. Repeat or severe violations result in a ban without prior warning.
